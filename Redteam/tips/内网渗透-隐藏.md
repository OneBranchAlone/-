# 内网渗透-隐藏

## Before

### Chrome默认以无痕模式启动

- Chrome快捷方式→右键属性→快捷方式→目标(T)

- 添加 `-incognito`属性：

```
"C:\Program Files\Google\Chrome\Application\chrome.exe" -incognito
```
